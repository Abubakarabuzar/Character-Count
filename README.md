# Character Counter App

A simple Character Counter web application that counts characters, words, and spaces in real time as the user types.  
This project is built using **HTML, CSS, and JavaScript**.

# Features
- Real-time character counting
- Word count
- Space count
- Clean and responsive UI
- Works instantly without page reload

# Tech Stack
- **HTML** – Structure of the app  
- **CSS** – Styling and layout  
- **JavaScript** – Logic for counting characters and words  

---

# How to Use
1. Open the application in a browser  
2. Type or paste text into the textarea  
3. Instantly see:
   - Total Character
   - Remaining Character
